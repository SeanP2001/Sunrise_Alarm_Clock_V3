Detailed Description

Represents an application for controlling USB ports, derived from the App base class.

Constructor & Destructor Documentation

◆ USBApp()

USBApp::USBApp ( Hardware &  hardware)

Construct a new USBApp object.

Currently both USB ports are set to come on at 6 and turn off at 9. In future, these magic numbers will be handled by the settings struct.

Parameters
